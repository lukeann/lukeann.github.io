# 自媒体接单合作跟进系统 (PR Tracker)

这是一个用于记录、管理和更改自媒体（特别是小红书）接单与合作进度的网页工具。

## 📖 项目简介
本项目是一个纯前端的网页应用，旨在帮助自媒体博主更高效地追踪每一个合作/PR的进度。通过这个工具，你可以方便地记录品牌方信息、合作状态、发布时间等关键节点，告别混乱的表格管理。

## 🚀 如何使用
本项目基于纯前端技术栈构建，无需复杂的配置或安装环境。

1. 克隆或下载本仓库代码到本地。
2. 双击打开 `index.html` 文件，即可在浏览器中运行和使用该跟进表。

访问在线版本（已部署在 GitHub Pages）：[https://lukeann.github.io/](https://lukeann.github.io/)
